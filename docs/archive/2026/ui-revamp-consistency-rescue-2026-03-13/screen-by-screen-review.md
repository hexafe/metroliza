# Screen-by-screen consistency rescue checklist

## Main dashboard
- Full dark client area.
- Header/status/workflow in one visual family.
- Workflow cards clickable but calmer.
- Status panel informational (not control-like).
- Mitigate bright native menu mismatch.

## About
- Full dark shell.
- Duck image integrated on themed surface.
- Hierarchy: image > version > author > repo action.

## Release notes
- Full dark shell.
- History in cohesive dark surfaces.
- Strong hierarchy and spacing.

## Parsing
- Full dark shell.
- Helper contrast improved.
- Disabled states consistent with theme.

## Export
- Full dark shell.
- Reduce frame-in-frame heaviness.
- Info areas styled as informational panels.

## Characteristic Name Matching
- Full dark shell.
- Calmer database strip.
- Better contrast and role hierarchy.
- Example panel as proper informational panel.
- Button hierarchy:
  - Primary: Save match
  - Secondary: Edit/Delete
  - Tertiary: import/export/clear/close

## Review and Rename Data
- Full dark shell.
- Stronger title/subtitle contrast.
- Empty states clearly visible.
- Table/action area feels cohesive.

## CSV Summary
- Full dark shell.
- Stronger section hierarchy.
- Better helper contrast.
- Clear final action emphasis.

## Progress dialogs
- Full dark shell.
- Duck image integrated.
- Modern progress bar styling.
- Readable status text.
- Consistent cancel button.

## Data grouping
- Full dark shell.
- Lower helper/status section aligned with system.
- Reduce control heaviness.

## Data filtering
- Full dark shell.
- Strong helper contrast.
